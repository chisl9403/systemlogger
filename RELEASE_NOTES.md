# Release Notes

## Version 1.0.0 (2025-11-19)

### 🎉 首次发布

这是SystemLogger的首个正式版本，一个完全适配Android 15的系统监控应用。

### ✨ 主要功能

#### 实时监控
- CPU温度监测（模拟数据）
- GPU温度监测（模拟数据）
- 电池温度监测（真实数据）
- 外壳温度监测（基于电池温度推算）
- 电池电量显示
- 充放电流监测
- 屏幕亮度显示

#### 数据可视化
- 实时温度曲线图表
- 4条独立曲线（CPU/GPU/电池/外壳）
- 颜色区分便于识别
- 自动滚动显示最新数据
- 触摸交互（缩放、拖动）

#### 数据管理
- CSV格式数据导出
- 1秒采样间隔
- 精确时间戳记录
- 自动文件管理

### 🔧 技术特性

#### Android 15完全适配
- ✅ 目标SDK: Android 15 (API 35)
- ✅ 现代权限系统
- ✅ 前台服务类型配置
- ✅ 通知权限适配
- ✅ Edge-to-Edge显示
- ✅ Material Design 3
- ✅ FileProvider安全共享

#### 温度读取
采用5层fallback机制确保温度数据可用：
1. HardwarePropertiesManager API
2. Thermal Zone文件系统
3. 电池温度Intent广播（主要使用）
4. 温度推算
5. 模拟数据fallback

### 🐛 已知问题

1. **温度传感器访问限制**
   - 部分设备无法直接读取CPU/GPU温度
   - 已实现fallback机制使用模拟数据
   - 电池温度为真实读取

2. **权限要求**
   - Android 13+需要手动授予通知权限
   - 首次启动会请求必要权限
